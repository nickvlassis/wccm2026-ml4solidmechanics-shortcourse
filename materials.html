Software & Prerequisites

Participants are expected to bring their own laptops with the following software installed prior to the course. Installation instructions will be provided with the pre-course materials.

  • Python 3.10+ with pip or conda
  • PyTorch (latest stable release)
  • NumPy, SciPy, Matplotlib
  • Jupyter Notebook or JupyterLab
  • Additional package requirements listed in the course repository
